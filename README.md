# `wordpress2hugo` - Migrate from WordPress to Hugo

## Usage

Use a

``` shell
./wordpress2hugo --help

./wordpress2hugo wordpress.xml --output foobar.github.io
```

## Features

- All original URLs are preserved as aliases
- Written in Python 3
